# Spotify Music Genre Analysis

## Overview
This project analyzes over 114,000 Spotify tracks to compare music genres based on their audio characteristics. Using Python, I cleaned the dataset, explored relationships between audio features, and created visualizations to identify trends across genres.

## Objectives
- Compare Spotify music genres using measurable audio features.
- Analyze popularity, danceability, energy, valence, tempo, and song duration.
- Identify patterns and relationships through exploratory data analysis.
- Present findings using visualizations and a technical report.

## Tools & Technologies

## Key Findings
The analysis revealed meaningful differences in audio characteristics across music genres. Genres varied in popularity, danceability, energy, valence, tempo, and song duration. A correlation analysis showed that energy and loudness were strongly correlated, while popularity had relatively weak relationships with most audio features.
